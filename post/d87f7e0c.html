发表于2022-11-13|更新于2022-11-14|测试

标签插入

使用插件: hexo-asset-image

安装方式:

npm install https://github.com/CodeFalling/hexo-asset-image -- save

引用图片方式: {% asset_img 图片名.jpg 示例图片 %}

注意：

  1. 要把图片放到对应的文件夹中**

  2. 标签插入时一定要按着格式来！！
